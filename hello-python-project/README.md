# hello-python-project

This project contains a simple Python script that demonstrates basic output functionality.

## Purpose

The purpose of this project is to provide a minimal example of a Python script that prints a message to the console.

## Files

- `src/hello.py`: A Python script that prints "Hello Copilot".
- `requirements.txt`: A file for listing dependencies (currently empty).
- `README.md`: Documentation for the project.

## How to Run the Script

1. Ensure you have Python installed on your machine.
2. Navigate to the `src` directory.
3. Run the script using the following command:

   ```
   python hello.py
   ```

This will output:

```
Hello Copilot
```
